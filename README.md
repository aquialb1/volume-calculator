A simple console-based program that calculates the volume of different shapes.

A shape type is picked by the user via an options menu, with each shape numbered 1, 2, 3, and onwards. The 0 key exits the program.

When a shape is chosen its corresponding volume function is called, with the function itself found in the header file. The user then inputs the specified dimensions for said shape and the volume is calculated then printed. If one or more inputs are invalid an error message will be printed, and the function loops for the user to try again.
